# 乘法口诀练习小游戏

一个给儿童练习乘法口诀的静态网页小游戏。无需安装依赖，也无需启动后端服务，直接用浏览器打开 `index.html` 即可使用。

## 功能特点

- 大字显示当前题目：`A × B = 答案框`
- A、B 都在 1-9 范围内随机出题，且始终满足 `A <= B`
- 支持分别配置 A 和 B 的出题范围
  - 例如：A 范围 `1-2`，B 范围 `5-9`
- 支持触摸屏软键盘输入
- 支持实体键盘数字键 `0-9` 输入
- 输入后自动判断，不需要按确认键
- 答对后答案框变绿色，停留 1 秒后进入下一题
- 答错后答案框变红色，并展示正确答案，停留 3 秒后进入下一题
- 答对 / 答错有不同提示音
- 历史记录展示：
  - 正确：绿色对钩
  - 错误：展示正确算式、划线的错误输入、红色叉号
- 乘法表正确率热力图
  - 三角形排布，左侧 9 格高，后续列逐渐减少
  - 统计每道题最近最多 5 次的正确率
  - 档位：未做、0%、25%、50%、75%、100%
- 正确率区域、历史记录区域、出题范围区域都可以收起 / 展开
- 数据保存在浏览器本地，刷新页面后不会丢失

## 使用方式

直接双击或用浏览器打开：

```text

```

推荐使用 Chrome、Edge 或其他现代浏览器。

## 出题范围说明

页面下方的“出题范围”可以配置 A 和 B 的范围。

例如：

```text
A: 1 到 2
B: 5 到 9
```

则可能出现：

```text
1×5, 1×6, ..., 1×9
2×5, 2×6, ..., 2×9
```

如果设置的范围无法生成满足 `A <= B` 的题目，例如：

```text
A: 8 到 9
B: 1 到 3
```

页面会保留原来的有效范围，并提示当前范围不可用。

## 判题规则

每输入一个数字后都会自动判断。

示例：

- 题目是 `2×3=6`
  - 输入 `6`：立即判定正确
  - 输入 `0`：立即判定错误

- 题目是 `3×4=12`
  - 输入 `1`：暂不判定，继续等待下一位
  - 再输入 `2`：判定正确
  - 输入 `2`：立即判定错误
  - 输入 `1` 后再输入 `9`：判定错误，历史记录保存错误输入 `19`

答错后，答案框会显示正确答案，但历史记录仍会保存孩子实际输错的内容。

## 正确率统计

每个乘法算式单独统计最近最多 5 次结果。

例如 `3×7`：

- 只做过 1 次且正确：正确率 100%
- 只做过 1 次且错误：正确率 0%
- 最近 4 次中正确 3 次：正确率 75%
- 最近 5 次中正确 2 次：正确率 40%，颜色会归到相邻档位显示

颜色档位：

- 灰色：未做过
- 鲜红色：0%
- 橙色：25%
- 黄色：50%
- 蓝色：75%
- 绿色：100%

## 本地数据

练习数据使用浏览器的 `localStorage` 保存。

保存内容包括：

- 出题范围设置
- 历史记录
- 每道题最近最多 5 次的正确 / 错误统计

如果更换浏览器、清理浏览器数据，或使用无痕模式，历史记录可能会丢失。

## 适配说明

页面适配横屏和竖屏触摸屏：

- 竖屏：题目、键盘、设置、统计区域纵向排列
- 横屏 / 大屏：练习区和统计区并排展示
- 数字键盘按钮较大，适合儿童触摸点击

## 技术说明

本项目是纯静态网页：

- HTML
- CSS
- JavaScript
- Web Audio API 用于生成提示音
- localStorage 用于保存数据

不依赖任何第三方库。
